#### `actor` The `activity` MAY have an `actor` describing the person, organisation or process which initiated the activity. If present, the `actor`: * MUST have an `@id` which MUST be an HTTP URI identifying the `actor`. * MUST have a `@type` which MUST include one of these values from AS 2.0: *Application*, *Group*, *Organization*, *Person* or *Service*. * MAY have a name * MAY have an `ldp:inbox` which MUST be an HTTP URI
