#### `target` The `activity` MUST have an `target` describing the {{ .Get 0 }} which is the intended recipient of the activity. The `target` MUST have: * An `@id` which MUST be an HTTP URI identifying the {{ .Get 0 }}. * A `@type` which MUST have the value _Service_ from AS 2.0. * An `ldp:inbox` which MUST be an HTTP URI for the LDN inbox for the {{ .Get 0 }}
